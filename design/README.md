# Design references (репозиторий движка)

Служебные материалы для согласования интерфейса документации с брендом. Лежат **вне** пользовательского контента (`docs-app/public/content/`), чтобы не попадать в индекс поиска по статьям и не путаться с материалами для читателей.

- **`colors/`** — цветовая палитра в Markdown; при смене токенов обновляйте **`docs-app/src/App.css`**.
