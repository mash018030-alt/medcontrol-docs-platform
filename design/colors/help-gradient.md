# Градиент баннера релизов (Help_000)

Линейный градиент для фона блока «Новый релиз» на хабе новостей (секции MC Cloud / MC Mobile).

|        | HEX       |
|--------|-----------|
| Начало | `#FF4794` |
| Конец  | `#2194FF` |

В CSS: `--docs-news-banner-gradient-start` и `--docs-news-banner-gradient-end` в `docs-app/src/App.css`, фон `.docs-news-banner__bg`: `linear-gradient(125deg, …)`.

См. также `color-palette.md`.
