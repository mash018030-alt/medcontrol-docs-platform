# Синий (акцент интерфейса)

Образец-скрин (если добавите): **`accent-blue.png`** в этой папке `design/colors/`.

HEX: **`#4A69FF`** · RGB: `74, 105, 255`

Используется для активных пунктов сайдбара, кнопки «Скачать в PDF», ссылок в статьях и прочих синих акцентов. В CSS: `--docs-ui-accent` / `--docs-ui-accent-hover` в `docs-app/src/App.css`.

Тёмно-синий для заголовков — отдельно: **`dark-blue.md`** / при необходимости **`dark-blue.png`** здесь же.
