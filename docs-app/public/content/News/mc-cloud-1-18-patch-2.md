# MC Cloud 1.18 Patch 2

И снова доработали платформу: теперь для конкретной организации или ПАК в АРМ можно задать собственные контакты технической поддержки – телефон и Telegram. Если контакт не указан, будет использоваться общий номер или контакт вышестоящей организации. Это поможет пользователям быстрее связываться с нужной службой поддержки, без лишней переадресации.

Настройки можно просматривать, редактировать или сбрасывать с учётом прав доступа.
