# MC Mobile 1.19.0

Этот релиз будет особенно полезен сервисным инженерам: мы улучшили работу мобильного приложения при сбоях отправки осмотров.

## Новые возможности

Теперь ПАК анализирует отправку осмотров и показывает предупреждение, если возникают сбои – например, если осмотр не ушёл на сервер.

![image3.png](/content/News/MC%20Mobile/mc-mobile-1.19.0/images/image3.png)

Помимо этого приложение сохраняет отчёты об ошибках. Их можно экспортировать на ПАК через сервисное меню.

![image4.png](/content/News/MC%20Mobile/mc-mobile-1.19.0/images/image4.png)

Там же при необходимости можно быстро очистить очередь осмотров – раньше для этого требовалось переустанавливать приложение и заново регистрировать ПАК.

![image2.png](/content/News/MC%20Mobile/mc-mobile-1.19.0/images/image2.png)

Маленький бонус. Также мы подготовили техническую основу для отображения персонализированных сообщений на ПАК – например, поздравлений с днём рождения. Планируем добавить эту возможность в ближайшем релизе MC Cloud.

## Исправление ошибок

Устранили ошибку по отображению пароля в лог-файлах

Часть пароля пользователей могла отображаться в лог-файлах после входа в приложение. Исправили – сейчас все пароли полностью маскируются и в логах не видны.
