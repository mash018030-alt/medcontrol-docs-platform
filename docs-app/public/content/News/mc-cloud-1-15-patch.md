# MC Cloud 1.15 Patch

По следам релиза MC Cloud 1.15 выпустили внеплановое обновление и упростили добавление документов.

Было: при загрузке документа любой пользователь должен заполнить все обязательные поля. Это могло вызывать сложности – не всем понятны значения полей и источники данных. К тому же без заполнения документ нельзя было сохранить.

Стало: документ можно просто загрузить без заполнения полей. Он получает статус «Ожидает подтверждения» и направляется медработнику на заполнение и проверку. Это удобно как для обычных пользователей, так и для медработников без прав подтверждения.

Пользователи с правом подтверждения документа могут выбрать:

- быструю загрузку файла без заполнения;
- загрузку с заполнением и подтверждением.
