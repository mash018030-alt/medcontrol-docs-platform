# MC Cloud 1.13 Patch

## Получение списка осмотров через публичный API

Можно подключить внешнюю систему (например, 1С или корпоративный портал) к MedControl и автоматически получать список проведённых медосмотров работников.

Через API доступны основные данные по каждому осмотру:

- ФИО сотрудника;
- дата осмотра;
- тип осмотра;
- результат;
- информация о ПАК.

Медицинские данные по осмотрам через Open Api не предоставляются.

Можно настраивать фильтрацию по дате, типу осмотра, сотрудникам и другим параметрам.

Для подключения Open Api нужно отправить заявку через менеджера MedControl или администратора системы.

## Уведомления об осмотрах в Telegram

Реализовали автоматические уведомления от MedControl в Telegram — о завершении медосмотра и необходимости пройти повторный. В таком уведомлении более подробная информация об осмотре, чем в талоне. Также это может быть альтернативой талону, если он не печатается или у организации нет принтера.

В уведомлениях отображаем ключевые сведения: ФИО работника, дата и время осмотра, заключение медработника (если есть), а при назначении повторного осмотра — время его прохождения.

Также показываем замечания, которые обычно не помещаются на талоне – это поможет избежать повторения ошибок в последующих осмотрах (например, работник не снял очки, или есть технические проблемы).

Уведомления отправляем тем, у кого Telegram привязан к профилю MedControl – сделать это можно будет через ПАК. Для этого нужно: 

![image1.png](/content/News/MC%20Cloud/mc-cloud-1-13-patch/images/image1.png)

1.  На главном экране нажать  
    «Привязать Telegram».
    
    * * *
    
2.  Отсканировать автоматически сгенерированный QR-код.
3.  Открыть диалог с ботом в Telegram.

После этого Telegram-чат будет автоматически связан с профилем работника.
