# MC Cloud 1.18 Patch (1–2)

В этом небольшом обновлении добавили новое уведомление для случаев, когда у работника зафиксированы пары алкоголя, но сам осмотр не завершён.

Уведомление помогает быстрее обратить внимание на осмотры с выявленным алкоголем и не терять их среди других незавершённых осмотров. При совпадении нескольких событий по одному осмотру отправляется одно уведомление, без дублирования.
