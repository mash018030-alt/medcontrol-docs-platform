# Организации

## Создание дочерней организации

Пользователи с административным доступом могут создать дочернюю организацию в АРМ. Это можно сделать через раздел «Организации» и через карточку родительской организации.

### Создание через раздел «Организации»

Для создания через раздел нужно:

1.  Открыть раздел «Организации» и нажать кнопку «Добавить».

![image5.png](/content/images/admin/image5.png)

2.  Выбрать тип организации.

![image32.png](/content/images/admin/image32.png)

3.  Заполнить данные:

- выбрать родительский элемент – головную организацию;
- полное и краткое наименование организации;
- логотип (при наличии);
- реквизиты – ИНН, КПП, ОГРН;
- контакты – телефон, e-mail, индекс, юридический адрес;
- переключить в активное положение чекбокс «Тестовая», если нужно создать дочернюю организацию в тестовом режиме и родительская организация не является тестовой.

![image35.png](/content/images/admin/image35.png)

4.  Выбрать платежный профиль – какое юр.лицо оплачивать осмотры организации. По умолчанию наследуется платежный профиль головной организации. Этот пункт можно пропустить и добавить позже при редактировании в карточке уже созданной организации.

![image38.png](/content/images/admin/image38.png)

5.  Если компания медицинская – заполнить данные профиля медоператора. Если критерии SLA отличаются от значений по умолчанию, можно добавить дополнительную категорию обслуживания с необходимыми параметрами SLA. Профиль медоператора можно добавить позже, при редактировании карточки уже созданной организации.

![image17.png](/content/images/admin/image17.png)

6.  Если компания немедицинская, добавить профиль медобслуживания – какое юр.лицо будет обрабатывать осмотры организации. Этот пункт можно пропустить и добавить позже при редактировании. Если указать две медорганизации, первая будет помечена как Основная – внести изменения можно будет также при редактировании организации.

![image34.png](/content/images/admin/image34.png)

7.  Профиль точки выпуска – фактический адрес или GPS-координаты места прохождения осмотров (при необходимости). Этот пункт можно пропустить и добавить позже при редактировании.

![image22.png](/content/images/admin/image22.png)

8.  После завершения появится сообщение об успешном добавлении, а организация появится в общем списке, в разделе «Организации».

![image4.png](/content/images/admin/image4.png)

### Создание через карточку головной организации

Для создания дочерней организации в карточке родительского элемента нужно:

1.  В карточке справа нажать «Добавить» и выбрать «Организацию».
2.  Выбрать тип организации. Выбрать можно только типы, которые могут быть дочерними для родительского элемента.
3.  Далее откроется форма создания с действиями, аналогичными созданию организации через раздел «Организации». Единственное отличие – на этапе заполнения основных данных родительская организация будет выбрана автоматически.

## Доступные действия с карточкой организации

### Редактирование

По кнопке «Редактировать» можно внести изменения в структуру организации и отредактировать:

- основные данные организации;
- родительскую организацию;
- платёжный профиль;
- профиль медоператора (для медорганизации);
- профиль медобслуживания – изменить текущий, добавить новый или сменить Основную медорганизацию;
- профиль точки выпуска ПАК.

![image41.png](/content/images/admin/image41.png)

### Добавление

По кнопке «Добавить» можно создать:

- профиль медоператора – юр.лица, обрабатывающего осмотры данной организации;
- профиль медобслуживания – для медорганизации;
- профиль точки выпуска или новый ПАК;
- учетную запись нового пользователя, который будет зарегистрирован в этой организации;
- дочернюю организацию;
- подписку на уведомление о событиях данной организации.

![image19.png](/content/images/admin/image19.png)

### Деактивация

Кнопка «Деактивировать» переводит карточку организации в неактивный режим, при этом данные организации сохраняются в АРМ.

![image10.png](/content/images/admin/image10.png)

Появляется сообщение об успешной деактивации, статус организации меняется на «Деактивирована».

![image29.png](/content/images/admin/image29.png)

В списке доступных действий кнопка «Деактивировать» заменится на «Активировать» – с её помощью можно восстановить активный статус организации в любой момент.

![image33.png](/content/images/admin/image33.png)

### История изменений

Также пользователям с административным доступом в карточке организации отображается вкладка «История изменений» – там можно увидеть, какие действия в организации были совершены, когда и кем.

![image11.png](/content/images/admin/image11.png)

## Настройки организации

Пользователи с административным доступом могут не только просматривать настройки организации, но и менять их.

![image43.png](/content/images/admin/image43.png)

### Основные настройки

На вкладке «Основные настройки» можно изменить формат табельного номера, способы авторизации работников на ПАК и виды осмотров, доступных для прохождения на ПАК работникам организации.

### Формат табельного номера

Для изменения формата табельного номера на ПАК нужно:

1.  На вкладке «Основные настройки» в блоке «Табельный номер» выбрать «Простая настройка» и нажать на иконку редактирования.
2.  В открывшемся окне внести изменения:

1.  Настройка минимального количества символов в табельном номере.
2.  Настройка максимального количества символов в табельном номере.
3.  Отметка «Содержит только цифры» не допускает использования букв и/или символов в формате табельного номера.

![image20.png](/content/images/admin/image20.png)

3.  Нажать «Сохранить», ознакомиться с предупреждением о том, что настройки применятся и к дочерним организациям, кликнуть «Да, я хочу внести изменения».

![image26.png](/content/images/admin/image26.png)

При редактировании вкладки «Продвинутая настройка» можно указать регулярное выражение, чтобы задать сложное правило валидации табельного номера.

![image23.png](/content/images/admin/image23.png)

Вернуть параметры к изначальным можно с помощью кнопки «Сбросить настройки».

![image2.png](/content/images/admin/image2.png)

### Виды осмотров

В блоке «Виды осмотров» можно управлять видами медицинских осмотров, которые работники организации могут проходить с ПАК.

По умолчанию установлены осмотры:

- предрейсовые;
- послерейсовые:
- медицинские осмотры в течение рабочего дня (смены).

Дополнительно доступны:

- предсменные;
- послесменные;
- алкотестирование – определение концентрации этанола (алкоголя) в выдыхаемом. Заключение по алкотестированию выносится автоматически, без участия медработника.

Для настройки видов осмотров нужно:

1.  В блоке «Виды осмотров» нажать на иконку редактирования.

![image42.png](/content/images/admin/image42.png)

2.  Выбрать необходимые виды осмотров (удалить ненужные/добавить новые). При перемещении карточки вида осмотра (выше/ниже) будет меняться последовательность видов осмотра на ПАК – согласно указанной в настройках. Нажать «Сохранить» и «Да, хочу внести изменения».

![image36.png](/content/images/admin/image36.png)

Кнопка «Сбросить настройки» возвращает виды доступных осмотров к настройкам по умолчанию.

![image53.png](/content/images/admin/image53.png)

### Способы авторизации на ПАК

Для изменения способов авторизации работников на ПАК нужно:

1.  В блоке «Способы авторизации на ПАК» нажать на иконку редактирования.

![image16.png](/content/images/admin/image16.png)

2.  В открывшемся окне изменить настройки авторизации.

1.  При перетягивании карточки способа авторизации будет меняться последовательность способов авторизации на ПАК – согласно последовательности, указанной в настройках.
2.  Кнопка Х удаляет способ авторизации на ПАК – пользователям организации не будет доступен удалённый способ авторизации.

![image50.png](/content/images/admin/image50.png)

3.  Нажать «Сохранить» и «Да, хочу внести изменения».

![image56.png](/content/images/admin/image56.png)

4.  Кнопка «Сбросить настройки» возвращает способы авторизации к настройкам по умолчанию.

![image21.png](/content/images/admin/image21.png)

### Гостевой доступ

Здесь можно настроить ПАК для прохождения медицинских осмотров работниками сторонней организации.

Для этого нужно:

1.  На вкладке «Гостевой доступ» нажать на кнопку «Изменить» или «Управлять» (если до этого настроек не было).

![image24.png](/content/images/admin/image24.png)

2.  В открывшемся окне выбрать организацию, работники которой смогут проходить осмотры на ПАК.

![image49.png](/content/images/admin/image49.png)

3.  Нажать «Сохранить».

![image47.png](/content/images/admin/image47.png)
