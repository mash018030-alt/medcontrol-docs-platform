# Новости и релизы (контент)

Структура сознательно близка к разделам в `0_docs/`: у каждой линии продукта своя папка, внутри — **`articles/`** и **`images/`**.

| Папка | Назначение |
|--------|------------|
| **`0_mc_cloud/`** | Релизы MC Cloud: статьи и иллюстрации для сайта. |
| **`1_mc_mobile/`** | Релизы MC Mobile: статьи и иллюстрации для сайта. |
| **`archives/`** | Выгрузки из Google Docs: ZIP, HTML, PDF и вложенные папки экспорта. Не считаются «источником правды» для публичных путей; после настройки движка PDF для кнопки «Скачать» лучше генерировать из актуальных `.md`. |
| **`news_tree.json`** | Дерево навигации (заголовки, логические пути `news/...`, при необходимости ссылки на PDF в `archives/`). |

Публичные пути к картинкам в markdown: `/content/1_news/0_mc_cloud/images/<slug_релиза>/<файл>.png` и аналогично для `1_mc_mobile`.
