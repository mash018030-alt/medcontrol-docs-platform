# Отчёты

В данном разделе можно создавать и анализировать различные виды отчётов, которые помогают контролировать итоги медицинских осмотров, финансовую информацию, оценку рисков среди работников и учёт временно отстранённых от выполнения трудовых обязанностей.

## Виды отчётов

На данный момент доступны следующие отчёты:

- медицинский журнал;
- биллинговый отчёт;
- группы риска;
- журнал регистрации отстраненных работников;
- журнал учета выданных справок в ЛПУ;
- журнал учета выданных направлений на мед. освидетельствование;
- статистика обработки осмотров.

### Медицинский журнал

Содержит данные о проведённых медицинских осмотрах работников с вынесенным медработником заключением. Фиксирует прохождение работниками следующих осмотров:

- предрейсовый осмотр;
- послерейсовый осмотр;
- предсменный осмотр;
- послесменный осмотр;
- осмотр в течение рабочего дня (смены);
- алкотестирование.

Меджурнал можно использовать для проверки выполнения обязательных медицинских осмотров перед допуском работников к выполнению трудовых обязанностей.

### Журнал регистрации отстраненных работников

Позволяет отслеживать причины отстранений, так как содержит информацию о работниках, которые по итогам осмотра были временно отстранены от выполнения трудовых обязанностей.

Может быть использован для подтверждения соблюдения всех процедур в отношении отстраненных работников, при проведении внутреннего аудита или проверки со стороны регулирующих органов.

### Статистика обработки осмотров

Отчёт предназначен преимущественно для медицинских администраторов.

Он содержит ключевые показатели эффективности работы медицинской организации по закрытию осмотров – учитываются суммарно обработанные осмотры всеми медработниками медицинской организации.

Как выглядит:

![15_statistika_obrabotki_osmotrov.png](/content/0_docs/1_obshee/images/17_otchety/15_statistika_obrabotki_osmotrov.png)

### Статистика смен

Отчёт предназначен преимущественно для медицинских администраторов.

Он дублирует информацию из вкладки «Смены» в карточке медработника и содержит сводку по рабочим сменам медработников: расписание и длительность смен, количество проведённых осмотров и ключевые показатели эффективности. Отчёт можно сформировать за нужный период по заданным медорганизациям и медработникам.

Как выглядит:

![16_statistika_smen.png](/content/0_docs/1_obshee/images/17_otchety/16_statistika_smen.png)

### Длительность обработки осмотров

Отчёт содержит сводные показатели длительности обработки осмотров на ключевых этапах – от взятия в работу до вынесения заключения.

Состоит из двух вкладок:

- успешные осмотры;
- неуспешные осмотры.
