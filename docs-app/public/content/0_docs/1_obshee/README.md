# Раздел «Общее» (контент)

Структура такая же, как у «Администрирование» (`0_docs/2_admin/`), «Медкабинет» (`0_docs/3_medkabinet/`) и «Медадминистратор» (`0_docs/4_medadmin/`).

- **`articles/`** — статьи с префиксами `00–25` в имени файла; оглавление — `00_main.md`.
- **`user_guide.md`** (в корне раздела) — та же разводящая, что и `articles/00_main.md`, для URL вида `/0_docs/1_obshee/user_guide` в движке; порядок и смысл блоков должны совпадать с разводящей в навигации движка (`docs-app/src/data/nav.js`), а не с устаревшим прототипом в `references/Index.tsx`.
- **`images/`** — иллюстрации по одной подпапке на статью (`01_avtorizaciya_v_arm`, …). Внутри только файлы, без вложенных каталогов. Имена как в разделе «Администрирование»: префикс порядка и транслитерация смысла, например `01_vkladka_osnovnye.png`, `02_tabnyj_nomer_okno.png`. Путь в Markdown: `/content/0_docs/1_obshee/images/<подпапка>/<файл>.png`.

Публичный URL страницы задаётся движком (часто совпадает с путём к `.md` без расширения).

Каталог **`images/obshee/`** в корне репозитория **не используется** для этого раздела после переноса. Служебные скриншоты плитки FAQ — в `images/25_chastye_voprosy/` (`faq_*.png`), описание — `README_faq_tile_screens.md` там же.
