# Раздел «Медадминистратор» (контент)

Руководство для медицинских администраторов в АРМ. Структура такая же, как у раздела документации «Администрирование» (`0_docs/2_admin/`).

- **`articles/`** — статьи с числовыми префиксами в имени файла; оглавление — `00_main.md`.
- **`images/`** — иллюстрации по подпапкам на статью (`01_nastroyki_medorganizacii`, …). В markdown пути вида `/content/0_docs/4_medadmin/images/<подпапка>/<файл>.png`.

Публичный URL страницы задаётся движком (обычно совпадает с путём к `.md` без расширения).
