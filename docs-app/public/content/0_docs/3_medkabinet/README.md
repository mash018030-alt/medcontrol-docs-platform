# Раздел «Медкабинет» (контент)

Структура такая же, как у разделов «Администрирование» (`0_docs/2_admin/`) и «Медадминистратор» (`0_docs/4_medadmin/`).

- **`articles/`** — статьи с числовыми префиксами в имени файла; оглавление — `00_main.md`.
- **`images/`** — иллюстрации по подпапкам (по одной на статью, где есть рисунки). В Markdown пути вида `/content/0_docs/3_medkabinet/images/<подпапка>/<файл>.png`.

Публичный URL страницы задаётся движком (часто совпадает с путём к `.md` без расширения).

Иллюстрации лежат **в этом разделе**, в `images/<подпапка>/`. Старый каталог **`images/medkabinet/`** в корне репозитория **удалён** после переноса файлов сюда.

Файл **`IMAGE_LEGACY_MAP.tsv`** сохраняет соответствие старых имён `imageNN.png` новым путям (удобно, если снова подтягиваете картинки из HTML-экспорта).
