# Подпись

Данный раздел находится в личном кабинете медработника в правом верхнем углу.

![Раздел «Подпись» в личном кабинете](/content/0_docs/3_medkabinet/images/08_podpis/01_razdel_podpis_menyu.png)

В разделе с подписями доступны:

1.  Просмотр данных сертификата УКЭП медицинского работника, выбор нужного сертификата УКЭП.

    ![Данные и выбор сертификата УКЭП](/content/0_docs/3_medkabinet/images/08_podpis/02_vkladka_podpisi_sertifikat.png)

2.  Контекстные подсказки в случае ошибок при подписи сформированных документов.

    ![Контекстные подсказки при ошибках подписи](/content/0_docs/3_medkabinet/images/08_podpis/03_podskazki_oshibok_podpisi.png)

Исходящие медицинские документы формируются автоматически в электронном виде и подписываются УКЭП медработника.
