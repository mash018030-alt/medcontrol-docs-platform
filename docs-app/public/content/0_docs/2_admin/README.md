# Раздел «Администрирование» (контент)

- **`articles/`** — статьи с префиксами `01–10` для порядка в дереве репозитория. Публичный путь к странице на сайте задаётся движком (обычно совпадает с путём к `.md` без расширения).
- **`images/`** — иллюстрации по подпапкам на статью (`02_organizacii`, `08_pak`, …). В markdown пути вида `/content/0_docs/2_admin/images/<подпапка>/<файл>.png`.

После этой раскладки каталог **`images/admin/`** в корне репозитория **не используется** для раздела документации «Администрирование».
