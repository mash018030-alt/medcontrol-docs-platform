# Референсы для сверки документации

Сюда при архивации/релизе можно класть официальные PDF-руководства как референс для отображения и сверки контента.

## Общее

- **Файл:** `1. Общее. MC Cloud. Руководство пользователя.pdf`
- **Назначение:** официальное руководство пользователя раздела «Общее»; контент статей в `public/content/obshee/` сверяется с этим PDF.
- Рекомендуется класть этот PDF в архив (например, в папку `reference/` или `docs/reference/`) при следующем релизе, чтобы всегда была эталонная версия для сравнения.
