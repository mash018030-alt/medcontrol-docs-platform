# Документы

Раздел «Документы» предназначен для поиска и просмотра входящих и исходящих документов – результаты осмотров, справки, обращения и пр.

## Поиск документов

В разделе можно искать документ по:

1. Виду – Входящий/Исходящий.
2. Названию.
3. Статусу.
4. Дате.
5. Расширенному фильтру.

![image99.png](/content/images/obshee/image99.png)

В фильтре по дате можно указать:

1. Готовый вариант диапазона.
2. Даты для произвольного периода поиска.
3. Время начала и окончания фильтрации.

![image51.png](/content/images/obshee/image51.png)

Через расширенный фильтр доступен поиск документа по:

1. Типу.
2. Работнику, в карточку которого загружен документ.
3. Организации работника, в карточку которого загружен документ.
4. Медработнику, загрузившему/обработавшему документ.

![image184.png](/content/images/obshee/image184.png)

В общем списке документов в разделе отображаются:

1. Выбранные для поиска параметры.
2. Дата загрузки документа.
3. Название документа.
4. Тип документа.
5. Статус.
6. ФИО работника, в карточку которого загружен документ.
7. Организация работника.

![image197.png](/content/images/obshee/image197.png)

## Карточка документа

Нажав на запись в общем списке, можно попасть в карточку документа с детальной информацией. Там отобразятся две вкладки – «Информация» и «История обработки».

### Информация

1. Вид документа.
2. Название документа.
3. Дата, время и часовой пояс формирования документа.
4. Информация об осмотре, по которому сформирован документ (для исходящих) или по работнику (для входящих). При клике на мини-карточку осмотра внутри исходящего документа откроется модальное окно с подробной информацией.
5. Для исходящих документов – другие документы по осмотру (при наличии).

![image63.png](/content/images/obshee/image63.png)

1. Статус документа.
2. Тип документа.
3. Номер и ссылка на осмотр, по которому был сформирован документ (для исходящих).
4. Работник, в карточку которого загружен документ.
5. ФИО медработника, подписавшего исходящий документ.
6. Кнопка для скачивания документа.

![image181.png](/content/images/obshee/image181.png)

7. Окно, где можно посмотреть документ, увеличить его, открыть в отдельном окне и скачать.

![image122.png](/content/images/obshee/image122.png)

### История обработки

На вкладке «История обработки» отображается вся информация по документу: кто и когда его добавил, изменил, утвердил или отправил на подпись. Также показано, использовался ли входящий документ для снятия блокировки с пользователя.
