# ПАК

Вкладка «ПАК» предназначена для просмотра всех программно-аппаратных комплексов, зарегистрированных в организации. Здесь можно искать ПАК, просматривать состояние, подключённые устройства и экспортировать логи работы.

## Поиск ПАК

В поиске можно быстро найти ПАК по:

1.  Названию.
2.  Статусу.
3.  Признаку «Тестовые» – для поиска ПАК, созданных в тестовой организации.
4.  Организации, статусу подключения и типу ПАК – через расширенный фильтр.

![image106.png](/content/images/obshee/image106.png)

В разделе со списком ПАК отображаются:

1.  Организация ПАК.
2.  Серийный номер ПАК.
3.  Тип ПАК – настольный/мобильный.
4.  Статус – подключен/отключен.
5.  Дата создания ПАК в АРМ.

Наверху отображается организация, в которой ПАК зарегистрирован, визуальное отображение типа ПАК – настольный/мобильный.

### Действия с ПАК

Справа от вкладок отображаются доступные пользователю действия с ПАК:

1.  Включить или отключить автоматическую печать талонов с результатами осмотров.
2.  Красная иконка рядом с кнопкой обозначает, что у ПАК нет принтера.
3.  Открыть список осмотров, пройденных на этом ПАК.
4.  Перейти на сайт метрологии ФГИС АРШИН с информацией о поверке ПАК.
