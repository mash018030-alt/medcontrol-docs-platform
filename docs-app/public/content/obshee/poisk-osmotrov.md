# Поиск осмотров

Для поиска необходимого медосмотра есть поле для ручного ввода данных и фильтры.

По умолчанию список осмотров упорядочен по дате проведения осмотра в порядке убывания: новые записи вверху, старые – внизу.

![image143.png](/content/images/obshee/image143.png)

- <span class="docs-li-underline">Поиск осмотра по его ID номеру:</span>

<details class="docs-disclosure">
<summary>Диапазон отображения осмотров по умолчанию</summary>
<div class="docs-disclosure-content">

При переходе в раздел «Осмотры» автоматически показываются осмотры за последние 24 часа. При клике на поле «За последние сутки» можно дополнительно отфильтровать осмотры по началу и концу периода и выбрать:

1. Автоматически заданный диапазон.
2. Диапазон по датам в календаре.
3. Время начала и окончания фильтрации.

![image17.png](/content/images/obshee/image17.png)

</div>
</details>
