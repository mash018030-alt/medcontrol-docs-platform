
| Термин | Определение |
| --- | --- |
| АД | Артериальное давление |
| Админпанель | История совершённых в АРМ действий |
| Активация | Процесс перевода чего-либо в активный режим |
| Алкотестирование | Процедура количественного определения алкоголя в выдыхаемом воздухе |
| АРМ | Автоматизированное рабочее место. Личный кабинет пользователя в WEB-версии платформы MedControl |
| Ассистент | Виртуальный помощник, анализирующий данные поступающих осмотров. Даёт медицинскому работнику предварительные рекомендации по заключению осмотров (допустить/не допустить) |
| Валидация | Процесс проверки полученных данных на соответствие заданным критериям |
| Верификация | Подтверждение соответствия представленной информации требованиям или установленным условиям |
| Входящие документы | Справки о состоянии здоровья и другие документы, загружаемые в систему пользователями |
| Гостевой осмотр | Осмотр работника на ПАК, не принадлежащем организации работника |
| Гостевые доступы | Доступ к использованию ПАК сторонними организациями |
| Группа разрешений | Набор отдельных разрешений, который определяет действия пользователя в АРМ MedControl |
| Деактивация | Перевод в неактивный режим (организации, пользователя и т.д.) |
| Дочерняя организация | Отдельное юридическое лицо, ИП или подразделение, которое принадлежит и контролируется материнской организацией, но действует независимо от неё |
| ЕРП | Единый реестр перевозчиков, государственная информационная система, содержащая данные об организациях, занимающихся пассажирскими или грузовыми перевозками |
| Заключение по осмотру | Документ, содержащий сведения об осмотре, принятом по нему решении медработника, а также данные об УКЭП медработника и лицензии медицинской организации |
| ИДС | Информированное добровольное согласие. Документ, подтверждающий получение работником необходимой информации и выражение согласия на проведение медицинских мероприятий |
| Иммобилизация | Обеспечение полной или частичной неподвижности одной или нескольких частей тела при повреждении |
| Импорт | Автоматическое или полуавтоматическое добавление данных из внешних источников в текущий файл/документ/базу данных |
| Исходящая медицинская документация | Документы, генерируемые системой на основании результатов осмотра. Подписываются УКЭП медработника |
| ИС | Информационная система |
| Код доступа | Код, который получает работник в своей организации для первого входа на ПАК и установки личного пароля |
| Контрагент | Организация (ЮЛ или ИП), оплачивающая услуги по проведению осмотров |
| ЛПУ | Лечебно-профилактическое учреждение |
| Медицинская организация | Организация с лицензией на ведение медицинской деятельности. В системе MedControl работники такой организации могут обрабатывать осмотры |
| Медицинский оператор | Медицинская организация, обрабатывающая осмотры в системе MedControl |
| MИ | Медицинское изделие |
| Мобильный ПАК | Мобильный программно-аппаратный комплекс для прохождения осмотров. Состоит из мобильного телефона/планшета и дополнительных устройств – СМК и принтера для печати талонов (опционально) |
| мм рт ст | Миллиметры ртутного столба, единица измерения артериального давления |
| Настольный ПАК | Программно-аппаратный комплекс, состоящий из металлического корпуса с планшетом, СМК и периферийным хабом (в который входят дополнительная внешняя камера и опционально – принтер для печати талонов) |
| НИИАТ | Научно-исследовательский институт автомобильного транспорта |
| Организация (орг.структура) | Организации, в том числе медицинские, а также группы организаций и подразделения |
| Организация-родитель | Организация, которая может структурно контролировать и управлять дочерними организациями |
| Осмотр | Медицинский осмотр, пройденный работником на ПАК |
| ПАК | Программно-аппаратный комплекс, на котором работники проходят осмотры |
| Параорбитальная область | Зона поверхности кожи вокруг глаз, включающая нижнее и верхнее веко, глазные щели, а также конъюнктивальный мешок и весь слезной аппарат |
| Периферийный хаб | Внешние дополнительные устройства, входящие в состав настольного ПАК (дополнительная камера, принтер для печати талонов и пр.) |
| ПК | Персональный компьютер |
| Платёжный профиль | Организация, оплачивающая осмотры (свои или организации) |
| Попытка | Отдельный медицинский осмотр в рамках серии предрейсовых или предсменных осмотров |
| Профиль медобслуживания | Настройка, определяющая, какие медицинские организации будут выносить заключения по осмотрам |
| Профиль медоператора | Настройка, дающая организации право обрабатывать осмотры и добавлять пользователей с типом роли «Медработник» |
| Работник | Пользователь системы, проходящий осмотры с ролью «Работник» |
| Референсные значения | Диапазон значений, в пределах которого результаты исследований считаются нормой для здорового человека |
| Роль | Роль, определяющая доступы пользователя к совершению тех или иных действий в АРМ |
| Сервисный инженер | Технический специалист, имеющий доступ к сервисному меню ПАК |
| Серия осмотров (сиквенс) | Последовательность до трёх предрейсовых или предсменных осмотров, назначаемых при выходе показателей (АД, пульс, температура) за допустимые границы |
| СМК | Система медицинского контроля – составная часть ПАК, медицинское устройство, которое позволяет производить тонометрию, термометрию и алкотестирование |
| Табельный номер | Уникальный номер работника, который выдаётся его организацией |
| Талон | Бумажный документ с заключением по результатам осмотра работника |
| Термометрия | Процедура измерения температуры тела |
| Токен | Защищенный носитель ключа электронной подписи |
| Точка выпуска | Физическое место, где установлен и используется настольный ПАК |
| Текущий медицинский осмотр | Медицинский осмотр в течение рабочего дня (смены) |
| Тонометрия | Процедура измерения артериального давления и частоты пульса |
| Уведомления | Подписка на нужные пользователю события организации |
| УЗ | Учётная запись (в АРМ) |
| УКЭП | Усиленная квалифицированная электронная подпись |
| УЦ | Удостоверяющий центр |
| ФГИС АРШИН | Государственная система Росстандарта для хранения и проверки сведений о поверке средств измерений, заменяющая бумажные свидетельства |
| ЧСС | Частота сердечных сокращений |
| ЭП | Электронная подпись |
| ЭПЛ | Электронный путевой лист |
| Этиология | Причины и условия возникновения заболевания, травмы |
| Целевые показатели | Настройки временных стандартов медицинской организации для взятия осмотров в работу и вынесения заключений |
| GPS-координаты | Широта и долгота, которые определяют положение точки относительно начала координат |
| CSV-файл | Текстовый формат, предназначенный для предоставления табличных данных. Файлы такого типа можно создавать и редактировать в Excel |
| ID | Уникальный номер в информационной системе |
| MedControl | Облачная платформа для автоматизации медицинских осмотров, обеспечивающая дистанционную передачу информации о состоянии здоровья работников и дистанционный контроль состояния их здоровья |
| MC | Сокращение от «MedControl» |
| MC Cloud | То же, что и АРМ. WEB-интерфейс платформы MedControl, предназначенный для просмотра информации об осмотрах, ПАК, организациях, пользователях, обработки осмотров и управления данными |
| MC Mobile | Мобильное приложение платформы MedControl, которое устанавливается на ПАК и отвечает за прохождение осмотров |
| RFID | Radio Frequency IDentification, радиочастотная идентификация – способ автоматической идентификации объектов. Смарт-карта (пластиковые карты со встроенной микросхемой), которая использует радиоволны для обмена информацией |
| RFID-карта | Карта, выданная работнику в своей организации для дальнейшей аутентификации на ПАК |
| SLA | Service Level Agreement. Соглашение об уровне обслуживания – требования, описывающие параметры предоставляемой услуги |
| t | Температура тела |
| °C | Градус Цельсия – единица измерения температуры |
