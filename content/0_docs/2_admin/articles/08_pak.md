# ПАК

## Создание ПАК

Создать новый ПАК можно:

- В разделе «ПАК» – кнопка «Добавить»:

![image15.png](/content/0_docs/2_admin/images/08_pak/01_razdel_dobavit_pak.png)

- В карточке организации по кнопке «Добавить» – «Добавить ПАК».

![image3.png](/content/0_docs/2_admin/images/08_pak/02_kartochka_dobavit_pak.png)

Если пользователю предоставлен доступ к нескольким организациям, при создании через раздел «ПАК» потребуется выбрать нужную организацию.

![image6.png](/content/0_docs/2_admin/images/08_pak/03_vybor_organizacii.png)

При создании через карточку ПАК организация указана автоматически.

![image45.png](/content/0_docs/2_admin/images/08_pak/04_organizaciya_avtomaticheski.png)

Далее нужно указать количество создаваемых ПАК и требуемый срок действия QR-кода. Этот код будет отсканирован камерой ПАК для сопряжения.

![image54.png](/content/0_docs/2_admin/images/08_pak/05_kolichestvo_pak_i_qr.png)

После нажатия кнопки «Добавить» появится карточка ПАК с временным названием ПАК и данными для сопряжения – QR-кодом, адресом сервера и id ПАК.

![image8.png](/content/0_docs/2_admin/images/08_pak/06_kartochka_pak_qr.png)

Несопряжённый ПАК можно найти в карточке организации на вкладке «ПАК» по фильтру «Ожидает сопряжения».

![image46.png](/content/0_docs/2_admin/images/08_pak/07_filtr_ozhidaet_sopryazheniya.png)

До окончания срока действия QR-код нужно отсканировать камерой нового ПАК для сопряжения. После сопряжения временное имя ПАК изменится на постоянное, появится статус «Подключен».

![image39.png](/content/0_docs/2_admin/images/08_pak/08_posle_sopryazheniya.png)

Также для сопряжения вместо сканирования QR-кода на ПАК можно ввести данные сервера и id ПАК.

![image40.png](/content/0_docs/2_admin/images/08_pak/09_vvod_servera_i_id.png)

## Действия с ПАК

Пользователям с административным доступом отображаются дополнительные доступные с ПАК действия:

1.  Запросить лог-файлы.
2.  Обновить поверку – запросить новые данные о поверке ПАК с сайта метрологии ФГИС АРШИН.
3.  Отвязать ПАК от организации.
4.  Деактивировать ПАК.

![image7.png](/content/0_docs/2_admin/images/08_pak/10_dejstviya_s_pak.png)

## Настройки ПАК

На вкладке «Настройки» в карточке ПАК можно просматривать и редактировать:

1.  Рекомендованную версию прошивки ПАК.
2.  Допустимое количество отказов от обновления.
3.  Режим печати талонов.

![image25.png](/content/0_docs/2_admin/images/08_pak/11_nastrojki_pak.png)

Для редактирования настроек нужно:

1.  На вкладке «Настройки» нажать на соответствующую иконку в нужном поле.

![image48.png](/content/0_docs/2_admin/images/08_pak/12_redaktirovanie_polya.png)

2.  Внести изменения и нажать «Сохранить».

![image9.png](/content/0_docs/2_admin/images/08_pak/13_sohranit_nastrojki.png)

При необходимости настройки можно быстро сбросить к значениям по умолчанию.

![image57.png](/content/0_docs/2_admin/images/08_pak/14_sbros_nastroek.png)

## История изменений

Также пользователям с административным доступом отображается вкладка «История изменений» в карточке ПАК. Там можно увидеть, какие изменения с ПАК произошли, кто их совершил и когда.

![image13.png](/content/0_docs/2_admin/images/08_pak/15_istoriya_izmenenij.png)
