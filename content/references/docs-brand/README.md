# Бренд платформы документации (контент)

Здесь — **человекочитаемый референс** по цветам и согласованию UI документации MedControl. Папка лежит в **`content/references/docs-brand/`**, не в движке: **базовая платформа** без продуктового бренда использует только нейтральные токены в **`engine/src/App.css`**.

- Папка **`references/`** при обходе контента для проверки ссылок **не** считается деревом статей сайта (как и **`brand/`**), поэтому эти материалы **не** попадают в навигацию и поиск по статьям.
- **`colors/`** — палитра в Markdown (таблицы HEX/RGB, пояснения). Точка входа — **`colors/color-palette.md`**.
- **Отображение в браузере (MedControl):** CSS‑переменные и шрифты — в **`content/brand/theme.css`** (подключается движком после `App.css`; см. **[`../../brand/README.md`](../../brand/README.md)**).
- **«Голая» платформа:** нейтральные дефолты — в **`engine/src/App.css`** (`:root`); продуктовую тему можно отключить через **`VITE_DOCS_USE_CONTENT_THEME=0`** (см. **[`../../brand/README.md`](../../brand/README.md)**).

**Логотипы, иллюстрации к статьям, PDF-эталоны** — в остальных ветках **`content/`** (например **`references/`**, **`0_docs/`**).
