# Цветовая палитра проекта

Используйте эти цвета в стилях и компонентах документации для единообразия.

**Вложения:** PDF и PNG от дизайна лежат **в этой же папке** (`content/references/docs-brand/colors/`), рядом с `.md` — например **`Colors big.pdf`**, **`Colors.png`**, **`sinii.png`**, **`temno_sinii.png`**, **`light.png`**, **`light-0.png`**.

| Название           | HEX      | RGB              | Использование                    |
|--------------------|----------|------------------|----------------------------------|
| Синий акцент (UI)  | `#4A69FF`| 74, 105, 255     | Активные пункты сайдбара, кнопка PDF, ссылки в статьях; CSS `--docs-ui-accent`; см. `accent-blue.md` |
| Тёмно-синий        | `#1A3399`| 26, 51, 153      | Заголовки статей, подчёркивание навигации при hover; см. `dark-blue.md` |
| Светлый (light)    | `#E6E9FF`| 230, 233, 255    | Плитки заглавных страниц, свёрнутый `summary` в `.docs-disclosure`, **фон объединённых и пустых ячеек** таблиц (`class="docs-table-cell-merged"`); CSS `--docs-light-bg`; см. `light.md` |
| Светлый 0 (light 0)| `#F5F6FF`| 245, 246, 255    | Более воздушный светлый фон; CSS `--docs-light-0-bg` в **`content/repo_docs/brand/theme.css`** (карточки главного дашборда и т.п.) |
| Важно / акцент     | `#CC8400`| 204, 132, 0      | Блок «Важно» (`.docs-important`), иконка, заголовок, рамка; см. [content-callouts.md](./content-callouts.md) |
| Внимание (розовый блок) | `#FCE4EC` фон, `#D81B60` акцент, `#F8BBD0` рамка | — | Макрос «Внимание» (`.docs-attention`); подробно в [content-callouts.md](./content-callouts.md) |
| Информация (нейтр.) | `#EFF6FF` фон, `#93C5FD` рамка | — | Макрос «Информация» (`.docs-info`); см. [content-callouts.md](./content-callouts.md) |
| Градиент релизов (Help_000) | `#FF4794` → `#2194FF` | 255, 71, 148 → 33, 148, 255 | Фон `.docs-news-banner__bg`; см. `help-gradient.md` |

Регламент классов, HTML-разметки и соответствия «Важно / Примечание / Внимание / Информация»: **[content-callouts.md](./content-callouts.md)**.

При добавлении новых цветов дополняйте таблицу здесь и переменные в **`content/repo_docs/brand/theme.css`** (продукт MedControl). Нейтральные дефолты «голой» платформы без контент-темы — только в **`engine/src/App.css`**.
