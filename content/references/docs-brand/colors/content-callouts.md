# Макросы (callout) в статьях документации

**Источник вёрстки:** селекторы в **`engine/src/App.css`** (структура под `.docs-article …`); **цветовые токены** макросов для MedControl — в **`content/brand/theme.css`** (`--docs-note-*`, `--docs-attention-*`, `--docs-important-*`, `--docs-info-*` и т.д.).  
**Краткий эталон для «Внимание»** также в Cursor-правилах: `.cursor/rules/medcontrol-docs-cursor-rules.mdc` (раздел про `.docs-attention`).

При смене цветов обновляйте **этот файл**, **`content/brand/theme.css`** и при необходимости нейтральные дефолты в **`App.css`** согласованно. В markdown статьях используется **сырой HTML** с фиксированными классами ниже — не подменяйте классы произвольными именами.

---

## Сводная таблица

| Название для читателя | Контейнер | Заголовок | Смысл | Основные цвета (HEX) |
|------------------------|-----------|-----------|--------|----------------------|
| **Важно** | `.docs-important` | `.docs-important-title` | Предупреждение / обязательное к учёту | Фон `#fdf8e4`, рамка и акцент `#CC8400` / `#cc8400` |
| **Примечание** | `.docs-note` | `.docs-note-title` | Пояснение; та же «золотая» линия, что у «Важно» | Токены `--docs-note-*` (фон `#fdf8e4`, рамка `#cc8400`, заголовок `#bf7a00`) |
| **Внимание** | `.docs-attention` | `.docs-attention-title` | Обязательные условия / ограничения (корп. **розовая** палитра; не путать с «Важно» и «Информация») | Токены `--docs-attention-bg` `#fce4ec`, `--docs-attention-accent` `#d81b60`, `--docs-attention-border` `#f8bbd0` |
| **Информация** | `.docs-info` | `.docs-info-title` | Нейтральная справка | Фон `#eff6ff`, рамка `#93c5fd`, иконка/акцент UI `--docs-ui-accent` (`#4A69FF`), заголовок `--docs-temno-sinii` |

Общая строка «Важно» в палитре бренда: [color-palette.md](./color-palette.md) (цвет `#CC8400`).

## Разметка в markdown (примеры)

**Важно**

```html
<div class="docs-important">
<div class="docs-important-title">Важно</div>
<p>Текст.</p>
</div>
```

**Примечание**

```html
<div class="docs-note">
<div class="docs-note-title">Примечание</div>
<p>Текст.</p>
</div>
```

**Внимание**

```html
<div class="docs-attention">
<div class="docs-attention-title">Внимание</div>
<p>Текст.</p>
</div>
```

**Информация**

```html
<div class="docs-info">
<div class="docs-info-title">Информация</div>
<p>Текст.</p>
</div>
```

Несколько абзацев — несколько `<p>`. Не смешивать классы с палитрами других макросов.

## Заметки для разработчиков

- Часть правил для `.docs-important` в начале `App.css` сидит в **длинной минифицированной первой строке** файла; остальные макросы продублированы в **читаемом блоке** ниже по файлу. При правке «Важно» проверяйте **оба места** (или вынесите «Важно» целиком в читаемый блок отдельной задачей).
- Для **«Внимание»** и **«Примечание»** менять отступы/скругления допустимо в том же блоке правил; для смены бренда у «Внимания» правьте **только** три переменные `--docs-attention-*` на `.docs-article .docs-attention` (как в правилах Cursor).
