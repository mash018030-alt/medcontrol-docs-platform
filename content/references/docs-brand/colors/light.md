# Светлый фон (плитки на заглавных страницах)

Образцы в этой папке: **`light.png`**, для оттенка «light 0» — **`light-0.png`** (раньше файл назывался `light 0.png`).

HEX фона: **`#E6E9FF`** · RGB: `230, 233, 255`

Используется как заливка карточек сетки разделов (`.docs-landing-tile`), **свёрнутой** полоски `summary` у раскрывающихся блоков `.docs-disclosure` и **объединённых / технически пустых ячеек** в таблицах статей (класс `docs-table-cell-merged` на `td`). В CSS (продукт): `--docs-light-bg` и `--docs-landing-tile-*` в **`content/brand/theme.css`**. Договорённость зафиксирована в `color-palette.md`.
