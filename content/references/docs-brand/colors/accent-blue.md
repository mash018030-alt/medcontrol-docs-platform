# Синий (акцент интерфейса)

Образец от дизайна в репозитории: **`sinii.png`** (то же значение, что у канонического имени **`accent-blue.png`**). Папка — **`content/references/docs-brand/colors/`**.

HEX: **`#4A69FF`** · RGB: `74, 105, 255`

Используется для активных пунктов сайдбара, кнопки «Скачать в PDF», ссылок в статьях и прочих синих акцентов. В CSS (продукт MedControl): `--docs-ui-accent` / `--docs-ui-accent-hover` в **`content/brand/theme.css`**; нейтральные дефолты платформы без темы — в **`engine/src/App.css`**.

Тёмно-синий для заголовков — отдельно: **`dark-blue.md`** / при необходимости **`dark-blue.png`** здесь же.
