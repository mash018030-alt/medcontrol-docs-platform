# Требования к техническим функциям платформы онлайн-документации MedControl

На основании слайда «Наблюдения: удачное» и блока «Какой видим идеальную онлайн-документацию Medcontrol» из отчёта.

---

## 1. Навигация по документации

**Основные элементы**

1. **Левое меню разделов**
   - древовидная структура документации
   - разделы → подразделы → статьи

2. **Оглавление статьи (Table of Contents)**
   - отображается справа
   - позволяет быстро переходить к разделам внутри статьи

3. **Поиск по документации**
   - глобальный поиск по всем статьям
   - поиск по названию и содержимому

4. **Навигация между статьями**
   - кнопки «Предыдущая статья» / «Следующая статья»

5. **Перекрёстные ссылки между статьями**
   - быстрый переход к связанным темам

---

## 2. Структура статьи

Статья должна поддерживать структурированные элементы.

**Обязательные элементы**
- Пошаговые инструкции с нумерацией
- Многоуровневые списки
- Раскрывающиеся блоки (accordion)

**Типы контента внутри статьи**
- изображения / скриншоты интерфейса
- таблицы
- схемы
- раскрывающиеся блоки

---

## 3. Визуальная подача

**Работа с изображениями**
- вставка скриншотов интерфейса
- выделение элементов стрелками или подсветкой
- возможность увеличения изображения

**Дополнительные визуальные элементы**
- таблицы
- схемы
- графические иллюстрации

---

## 4. Макросы и акцентные блоки

Специальные блоки-выделения:
- ⚠️ Внимание
- 💡 Совет
- 📝 Примечание

---

## 5. Дополнительные функции статьи

**Медиа**
- видео внутри статьи

**Обратная связь**
- блок «Была ли статья полезна?»
- возможность оставить отзыв

**Метаданные статьи**
- отображение даты последнего обновления

---

## 6. Экспорт и печать

- кнопка «Скачать PDF»
- кнопка «Печать статьи»
- PDF-версия: корректное отображение при печати, читаемая структура

---

## 7. Требования к UX (чего избегать)

- длинных «простыней» текста без структуры
- неочевидной навигации
- слишком крупных UI-элементов
- всплывающих элементов, мешающих чтению
- перегруженности макросами

---

## 8. Итоговый список фич для прототипа

**Навигация**
- левое меню разделов
- оглавление статьи
- поиск по документации
- переход «вперед / назад»
- перекрёстные ссылки

**Структура статьи**
- нумерованные шаги
- многоуровневые списки
- accordion-блоки

**Контент**
- изображения
- таблицы
- схемы
- видео

**Акценты**
- блоки: Важно / Совет / Примечание

**Системные функции**
- дата обновления
- оценка статьи
- печать
- экспорт PDF
