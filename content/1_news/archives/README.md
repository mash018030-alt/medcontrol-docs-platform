# Архивы выгрузок (Google Docs и т.п.)

Сюда перенесены материалы, которые приходили как **архивы/экспорты** (ZIP, HTML, PDF из подписной папки релиза), а не как отредактированные статьи для платформы.

- **`0_mc_cloud/`** — облачные релизы: ZIP-файлы и распакованные папки с HTML, PDF и исходным `images/` из экспорта.
- **`1_mc_mobile/`** — мобильные релизы: то же.

Рабочий контент для сайта лежит в `../0_mc_cloud/` и `../1_mc_mobile/` (`articles/` + `images/`). При добавлении нового релиза можно класть новый ZIP или папку сюда, затем переносить текст в `articles/` и картинки — в `images/<slug>/`.
