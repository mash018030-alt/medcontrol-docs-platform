# Бренд платформы документации (репозиторий движка)

Здесь — то, что описывает **организацию самой платформы документации** относительно бренда: прежде всего цвета и согласование UI. **Логотипы, иллюстрации к статьям, PDF-эталоны и прочий контент** лежат в **`content/`** (корень репозитория), а не в этой папке.

Материалы в `docs-brand/` держим **вне** публикуемых статей, чтобы они не попадали в поиск по документации и не смешивались с материалами для читателей.

- **`colors/`** — цветовая палитра в Markdown; при смене токенов обновляйте **`docs-app/src/App.css`**.
