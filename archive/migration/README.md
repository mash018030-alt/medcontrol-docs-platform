# Архив миграции

Здесь лежат **служебные снимки**, сделанные при разделении монорепозитория на движок (`medcontrol-docs-platform`) и репозиторий контента. На повседневную работу это не влияет.

| Файл | Зачем храним |
|------|----------------|
| [`baseline-links-before-migration.txt`](baseline-links-before-migration.txt) | Вывод `node docs-app/scripts/check-internal-links.mjs` и сводка `flatArticles` ↔ файлы `.md` на **2026-04-02**, до переноса контента в отдельный репозиторий. Нужен, чтобы при спорных регрессиях ссылок или навигации было с чем сравнить «как было на линии разреза». |

При необходимости новую базовую линию можно снова снять тем же скриптом и положить сюда с датой в имени файла.
